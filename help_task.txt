## task コマンド

- `get` タスクの取得
    - 登録している全タスクを表示します
    - `@BOT_yaranai task get`

- `add` タスクの追加
    - 新しくタスクを追加します
    - `@BOT_yaranai task add "Title" "Description" ConditionID Difficulty "DueDate"`
        - "Title" タスク名
        - "Description" タスクの詳細
        - ConditionID コンディション
            - `@BOT_yaranai condition get` で一番左の列に表示される ID
        - Difficulty タスクのコスト(ウェイト)
            - 1 軽い
            - 2 普通
            - 3 重い
        - DueDate タスクの期限
            - `yyyy-mm-dd`
                - `yyyy`: 年
                - `mm`  : 月
                - `dd`  : 日
    - 入力例
        - `@BOT_yaranai task add 醤油を買う 無いので 1 2 2099-12-31`
        
- `edit` タスクの編集
    - 現在登録されているタスクの内容を編集します
    - `@BOT_yaranai task edit TaskID "Title" "Description" ConditionID Difficulty "DueDate"`
        - TaskID 各タスク固有のID
            - `@BOT_yaranai task get` の一番左の列に表示される ID
        - "Title" タスク名
        - "Description" タスクの詳細
        - ConditionID コンディション
            - `@BOT_yaranai condition get` で一番左の列に表示される ID
        - Difficulty タスクのコスト(ウェイト)
            - 1 軽い
            - 2 普通
            - 3 重い
        - DueDate タスクの期限
            - `yyyy-mm-dd`
                - `yyyy`: 年
                - `mm`  : 月
                - `dd`  : 日
        - 変更しない要素は アンダーバー (`_`) を置く
    - 入力例
        - `@BOT_yaranai task edit 1 _ 醤油がないので _ 1 2100-01-01`

- `delete` タスクの削除
    - 現在登録されているタスクを削除します
    - `@BOT_yaranai task delete TaskID`
        - TaskID 各タスク固有のID
            - `@BOT_yaranai task get` の一番左の列に表示される ID
    - 入力例
        - `@BOT_yaranai task delete 1`
